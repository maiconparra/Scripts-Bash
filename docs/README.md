# Scripts-Bash

Ferramenta modular para instalação, remoção e gerenciamento de ambientes de desenvolvimento (PHP, Docker, bancos de dados, etc) via Bash.

## Estrutura do Projeto
- **bin/**: Scripts principais e menu interativo
- **modules/**: Scripts modulares para cada aplicação
- **utils/**: Funções utilitárias compartilhadas
- **docs/**: Documentação do projeto
- **.vscode/**: Configurações e recomendações para VS Code

## Instalação e Uso
Consulte os arquivos em docs/ para detalhes de instalação, empacotamento .deb, uso do menu e comandos CLI.
